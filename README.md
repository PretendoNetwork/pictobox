# Picto Box
TypeScript library for interacting with several different image types used by the Wii U and 3DS.

## Installation
```
npm i @pretendonetwork/pictobox
```

## Supported files
- [ ] BMP
- [ ] TGA
- [ ] PNG
- [ ] ETC1
- [ ] [Indexed images](https://github.com/PretendoNetwork/indexed-image-converter)
